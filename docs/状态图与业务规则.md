
## 发布物（资源/帖子）规则清单

## 0）状态

`Draft` 草稿 `PendingReview` 待审 `Reject` 拒绝 `Approved` 通过（待自动上架） `Listed` 可见 `Unlisted` 不可见 `Deleted` 删除（终态）

---

## 1）动作与跳转规

1. **创建**
    

- `∅ → Draft`（作者）
    

2. **编辑并保存（草稿态）**
    

- `Draft → Draft`（作者）
    
- 其他状态编辑：`ERR_STATE`
    

3. **下架态修改并保存（关键）**
    

- `Unlisted + 保存修改 → Draft`（作者）
    
- 语义：一旦修改内容，必须重新触发审核
    

4. **提交审核**
    

- `Draft → PendingReview`（作者）
    
- 否则：`ERR_STATE / ERR_PERM`
    

5. **撤回待审（重新编辑）**
    

- `PendingReview → Draft`（作者）
    
- 否则：`ERR_STATE / ERR_PERM`
    

6. **审核驳回**
    

- `PendingReview → Reject`（审核员/管理员；必填 reason）
    
- 否则：`ERR_STATE / ERR_PERM / ERR_REASON`
    

7. **修改后重提**
    

- `Reject → PendingReview`（作者）
    
- 否则：`ERR_STATE / ERR_PERM`
    

8. **审核通过**
    

- `PendingReview → Approved`（审核员/管理员）
    
- 否则：`ERR_STATE / ERR_PERM`
    

9. **自动上架**
    

- `Approved → Listed`（系统自动）
    

10. **下架**
    

- `Listed → Unlisted`（作者/管理员）
    
- 否则：`ERR_STATE / ERR_PERM`
    

11. **重新上架（无修改，直接恢复）**
    

- `Unlisted → Listed`（作者/管理员）
    
- 前置：下架后**未发生保存修改**（见规则3）
    
- 否则：`ERR_STATE / ERR_PERM`
    

12. **重新上架（已修改并保存，必须重审）**
    

- 触发方式：走规则3先变为 `Draft`
    
- 路径：`Unlisted(保存修改)→ Draft → PendingReview → Approved → Listed`
    
- 若用户试图在 Draft/PendingReview/Reject/Approved 直接“上架”：`ERR_STATE`
    

13. **作者删除**
    

- `Draft | PendingReview | Reject | Unlisted → Deleted`（作者）
    
- 否则：`ERR_STATE / ERR_PERM`
    
    - 注：作者**不可**删除 `Listed`
        

14. **管理员删除**
    

- `Listed → Deleted`（管理员）
    
- 可选：管理员也可删除任意状态
    
- 否则：`ERR_STATE / ERR_PERM`
    

15. **终态约束**
    

- `Deleted` 不允许任何动作（全部 `ERR_STATE`）
    

---

## 2）错误语义（最小集）

- `ERR_STATE`：当前状态不允许该动作（含“想上架但需重审”）
    
- `ERR_PERM`：无权限（非作者/非审核员/非管理员）
    
- `ERR_REASON`：驳回必填 reason
    
- `ERR_NOT_FOUND`：目标不存在或已删除

## 用户评论规则清单

## 0）状态

`Visible` 可见 `Deleted` 删除（终态）

---

## 1）动作与跳转规则

1. **发表**
    

- `∅ → Visible`（登录用户）
    
- 前置：目标发布物必须 `Listed`
    
- 否则：`ERR_CLOSED / ERR_PERM`
    

2. **删除（自己删）**
    

- `Visible → Deleted`（评论作者）
    
- 否则：`ERR_STATE / ERR_PERM`
    

3. **删除（发布物作者删）**
    

- `Visible → Deleted`（发布物作者）
    
- 否则：`ERR_STATE / ERR_PERM`
    

4. **删除（管理员删）**
    

- `Visible → Deleted`（管理员）
    
- 否则：`ERR_STATE / ERR_PERM`
    

5. **自动删除（机审命中）**
    

- `Visible → Deleted`（系统）
    

6. **终态约束**
    

- `Deleted` 不允许任何动作（`ERR_STATE`）
