# 页面清单（按业务流）

| route | 页面名 | 主要目标动作 | 关键模块(3-6个) | 入口→出口 |
|---|---|---|---|---|
| `/` | 首页 | 搜索与发现资源 | 顶部导航、搜索框、分类筛选、资源流卡片、分页/加载更多 | 站点直达/导航 -> 资源列表页或资源详情页 |
| `/prompts` | Prompts 列表 | 浏览与筛选 Prompt | 资源类型切换、排序筛选、Prompt 卡片、分页、空态 | 首页/导航 -> `/prompts/[id]` 或 `/prompts/new` |
| `/prompts/new` | Prompt 创建 | 发布新的 Prompt | 表单区、字段校验、标签/分类选择、提交栏、取消返回 | `/prompts` -> 提交后 `/prompts/[id]` 或返回列表 |
| `/prompts/[id]` | Prompt 详情 | 查看详情并触发互动 | 标题头、元信息、正文内容、操作栏（收藏/点赞）、相关推荐 | 列表/搜索 -> 编辑页或返回列表 |
| `/prompts/[id]/edit` | Prompt 编辑 | 修改并保存 Prompt | 编辑表单、变更提示、状态提示、保存按钮、取消按钮 | 详情页/我的发布 -> 保存后回详情 |
| `/skills` | Skills 列表 | 浏览与筛选 Skill | 资源类型切换、筛选条、Skill 卡片、分页、空态 | 首页/导航 -> `/skills/[id]` 或 `/skills/new` |
| `/skills/new` | Skill 创建 | 发布新的 Skill | 表单区、安装信息输入、标签/分类选择、校验提示、提交栏 | `/skills` -> 提交后 `/skills/[id]` 或返回列表 |
| `/skills/[id]` | Skill 详情 | 查看 Skill 并获取使用方式 | 标题头、安装说明、使用说明、操作栏、相关推荐 | 列表/搜索 -> 编辑页或返回列表 |
| `/skills/[id]/edit` | Skill 编辑 | 修改 Skill 信息 | 编辑表单、字段校验、状态提示、保存按钮、取消按钮 | 详情页/我的发布 -> 保存后回详情 |
| `/mcps` | MCPs 列表 | 浏览与筛选 MCP | 资源类型切换、筛选条、MCP 卡片、分页、空态 | 首页/导航 -> `/mcps/[id]` 或 `/mcps/new` |
| `/mcps/new` | MCP 创建 | 发布新的 MCP | 表单区、配置文本输入、分类标签选择、校验提示、提交栏 | `/mcps` -> 提交后 `/mcps/[id]` 或返回列表 |
| `/mcps/[id]` | MCP 详情 | 查看 MCP 配置与说明 | 标题头、配置区、使用说明区、操作栏、相关推荐 | 列表/搜索 -> 编辑页或返回列表 |
| `/mcps/[id]/edit` | MCP 编辑 | 修改 MCP 内容 | 编辑表单、字段校验、状态提示、保存按钮、取消按钮 | 详情页/我的发布 -> 保存后回详情 |
| `/tutorials` | 教程列表 | 浏览教程内容 | 筛选排序、教程卡片、标签区、分页、空态 | 首页/导航 -> `/tutorials/[id]` 或 `/tutorials/new` |
| `/tutorials/new` | 教程创建 | 发布新教程 | 富文本/正文输入、封面与元信息、标签分类、校验提示、提交栏 | `/tutorials` -> 提交后 `/tutorials/[id]` 或返回列表 |
| `/tutorials/[id]` | 教程详情 | 阅读教程并互动 | 标题头、正文内容、目录/锚点、操作栏（收藏/点赞）、相关推荐 | 列表/搜索 -> 编辑页或返回列表 |
| `/tutorials/[id]/edit` | 教程编辑 | 修改教程并保存 | 编辑表单、正文编辑区、状态提示、保存按钮、取消按钮 | 详情页/我的发布 -> 保存后回详情 |
| `/ranks` | 榜单页 | 查看各类排行榜 | 榜单切换、榜单卡片、时间维度筛选、排名信息、空态/加载态 | 顶部导航 -> 资源详情页或返回首页 |
| `/login` | 登录页 | 进入登录流程 | 登录表单、第三方登录区、错误提示、返回路径参数、提交按钮 | 受限入口/导航 -> 登录后回来源页 |
| `/me/favorites` | 我的收藏 | 管理收藏资源 | 用户导航、收藏列表、筛选排序、取消收藏操作、空态 | 个人中心入口 -> 资源详情页 |
| `/me/published` | 我的发布 | 管理已发布内容 | 用户导航、发布列表、状态标签、编辑入口、筛选区 | 个人中心入口 -> 详情页或编辑页 |
| `/admin` | 后台首页 | 查看后台总览并分发管理入口 | 后台导航、统计卡片、快捷入口、待办提醒、系统状态 | 后台入口 -> 各后台子模块 |
| `/admin/categories` | 分类管理 | 维护分类数据 | 分类列表、搜索筛选、新增/编辑弹层、启停开关、保存反馈 | 后台首页 -> 保存后留在本页 |
| `/admin/tags` | 标签管理 | 维护标签数据 | 标签列表、搜索筛选、新增/编辑弹层、状态开关、保存反馈 | 后台首页 -> 保存后留在本页 |
| `/admin/moderation/cases` | 审核案件列表 | 处理待审核内容 | 审核列表、状态筛选、案件摘要、批量操作、分页 | 后台首页 -> `/admin/moderation/cases/[case_id]` |
| `/admin/moderation/cases/[case_id]` | 审核案件详情 | 执行通过/驳回等审核动作 | 内容快照、审核意见输入、动作按钮、历史记录、返回列表 | 审核列表 -> 处理后回审核列表 |
| `/admin/events` | 事件日志 | 查看系统事件 | 日志列表、过滤器、事件详情抽屉、时间范围筛选、分页 | 后台首页 -> 查看后返回后台或留在本页 |
| `/admin/audit-logs` | 审计日志 | 审计关键操作记录 | 日志列表、操作人筛选、对象筛选、详情面板、导出入口 | 后台首页 -> 查看后返回后台或留在本页 |
| `/admin/users` | 用户管理 | 检索并管理用户 | 用户列表、搜索筛选、状态开关、批量操作、分页 | 后台首页 -> `/admin/users/[user_id]` |
| `/admin/users/[user_id]` | 用户详情 | 查看并调整用户状态/角色 | 用户资料区、角色权限区、操作历史、状态操作栏、返回列表 | 用户列表 -> 保存后回列表或停留详情 |
| `/admin/roles` | 角色管理 | 管理角色定义 | 角色列表、角色编辑、新增角色、成员关联、保存反馈 | 后台首页 -> 保存后留在本页 |
| `/admin/permissions` | 权限管理 | 管理权限项 | 权限列表、分组筛选、权限编辑、状态切换、保存反馈 | 后台首页 -> 保存后留在本页 |
| `/admin/role-permissions` | 角色权限矩阵 | 配置角色-权限映射 | 矩阵表格、角色选择、权限选择、批量勾选、保存按钮 | 后台首页/角色管理 -> 保存后回角色管理或留页 |
