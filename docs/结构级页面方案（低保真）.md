# 页面方案（结构级｜框架模版对齐版）

> 产出：**全站页面树** + **每页信息块（首屏/Tab/折叠/等）** + **块→字段映射** + **空态/错态/加载态** + **权限拦截与回跳**
> 
> 对齐你的框架模版：**首页大搜索**、**列表左侧排序/筛选**、**详情右侧元信息栏+评论置底**、**编辑页底部按钮区**、**个人中心两套左栏（收藏=文件夹树/发布=类型导航）**、**后台左栏菜单**

---

## 0) 全局约定（与已定 API / 权限一致）

### 0.1 顶部导航（全站一致）

- 左：Logo「AI资源站」→ **/**（网站首页）
- 中：**Prompt / Skill / MCP / 教程 / 榜单**
- 右：**个人中心**（已登录）/ 登录（未登录）

### 0.2 鉴权与拦截

- 未登录访问需登录页面：后端 **401** → 前端跳 **/login?returnTo=…**
- 已登录但权限不足：后端 **403** → 前端展示「权限不足」
- 内容可见性：
    - `Listed`：公开可见
    - 其他状态：作者/审核员/管理员可见；其他人访问 403

### 0.3 搜索与高亮

- 列表页搜索：在各列表页顶部搜索框内；参数 `q`，支持 `highlight=1`
- 首页搜索：全站搜索，**结果直接展示在首页搜索框下方（卡片混合流）**
- 高亮：列表/首页结果卡片显示 `highlight.title/one_liner/document`

### 0.4 “分类=标签”（UI口径合并；数据层仍区分）

- **前台筛选区只展示一套「标签」控件**（勾选多项）
- 参数映射：前台筛选统一走 `tag_ids=...`（多选）
- `category_id` 仍作为内容必填字段，用于编辑/发布与后台管理（不在前台筛选区单独出现）

## 2) 页面结构（按你的框架模版拆块）

> Block 只是结构级，不涉及视觉细节；同名 Block 全站复用。

### 2.1 HomePage（/）

- Block H1：顶部导航（全站）
- Block H2：主文案（可选）
- Block H3：**大搜索框**（q 输入 + 搜索按钮）
- Block H4：**搜索结果区（卡片流，混合 type）**
    - 空态：未搜索/无结果
    - 分页：offset/limit（可做“加载更多”）

### 2.2 ListPage（/prompts /skills /mcps /tutorials）

- Block L1：顶部导航（全站）
- Block L2：页内搜索框（q + 搜索按钮）
- Block L3：左侧栏
    - L3-1：排序区（hot_score/created_at/views_7d + order asc/desc）
    - L3-2：条件筛选区（**标签勾选** + source(仅 mcp/skill)）
- Block L4：内容区（卡片网格）
- Block L5：分页（offset/limit/total）
- Block L6：空/错/加载态容器

### 2.3 DetailPage（/type/{id}）

- Block D1：顶部导航（全站）
- Block D2：主内容区（左）
    - D2-1：详情主体（统一首屏信息 + Tabs）
    - D2-2：**评论区（固定在详情下方，不进 Tabs）**
- Block D3：右侧元信息栏（只放展示+轻动作）
    - 作者/发布时间
    - 点赞/收藏/分享（轻操作）
    - 统计（views_7d/up/comments/hot_score）
- Block D4：空/错/加载态容器

### 2.4 EditorPage（/type/new, /type/{id}/edit）

- Block E1：顶部导航（全站）
- Block E2：表单主体（信息填写区）
- Block E3：右侧元信息栏（校验提示/当前状态/最近保存）
- Block E4：底部按钮区（普通区域，滚到最底部才看到）
    - 保存 / 提交审核 / 上架 / 下架 / 删除
- Block E5：空/错/加载态容器

### 2.5 Me-Favorites（/me/favorites）

- Block M1：顶部导航（全站）
- Block M2：左侧个人中心导航（固定）
    - 项：收藏 / 发布
    - **收藏项可展开：文件夹树（嵌套在收藏项下方）**
- Block M3：右侧内容区
    - 当前文件夹信息 + 卡片流
- Block M4：文件夹操作（新建/重命名/删除）
- Block M5：删除文件夹二次确认（会取消该文件夹内全部收藏）

### 2.6 Me-Published（/me/published）

- Block P1：顶部导航（全站）
- Block P2：左侧个人中心导航（固定）
    - 项：收藏 / 发布
    - **发布项可展开：类型导航（Prompt/Skill/MCP/教程）**
- Block P3：右侧内容区（卡片流）
    - 状态筛选（Draft/PendingReview/Reject/Approved/Listed/Unlisted）
    - 搜索 q
    - 排序（hot_score/created_at/views_7d）
    - 分页
    - 操作：编辑/提交审核/上架/下架/删除（在卡片上或卡片菜单内）

### 2.7 AdminLayout（/admin/**）

- Block A1：后台顶栏（AI资源站 / 管理后台 / 个人中心）
- Block A2：左侧菜单（所有管理项均可找到入口）
    - 内容审核（cases）
    - 分类管理（categories）
    - 标签管理（tags）
    - 用户管理（users）
    - 权限管理（roles/permissions/矩阵）
    - 日志中心（events/audit-logs）
- Block A3：主工作区（表格/详情）

## 3) 块→字段映射（核心页面）

### 3.1 首页全站搜索（/）

**Block H3 大搜索框**

- 入参：`q`（输入框）
- 触发：点击搜索或回车 → 请求 `/search`

**Block H4 搜索结果区（混合卡片）**

- API（新增）：`GET /search?q=&offset=&limit=&highlight=1&sort=hot_score&order=desc`
- 返回：`ContentSummary[] + meta`
- 卡片字段：同 ContentSummary（含 type 徽标 + highlight 片段）

---

### 3.2 列表页（/prompts 等）

**Block L2 页内搜索**

- query：`q` + `highlight=1`

**Block L3-1 排序区**

- `sort=hot_score|created_at|views_7d`
- `order=asc|desc`（默认 desc）

**Block L3-2 条件筛选区**

- 「标签」多选：`tag_ids=...`
- 「source」(仅 mcp/skill)：`source=official|user`

**Block L4 卡片**

- `ContentSummary`：
- `title/one_liner/category/tags/author/cover_asset_id/stats_7d/created_at`
- `highlight.*`（开启高亮时展示）

**Block L5 分页**

- `meta.offset/meta.limit/meta.total`

---

### 3.3 详情页（/type/{id}）

**Block D2-1 首屏信息卡 + Tabs**

- 数据源：各类型详情接口
    
- Prompt：`GET /prompts/{id}` → `PromptDetail`
    
- MCP：`GET /mcps/{id}` → `McpDetail`
    
- Skill：`GET /skills/{id}` → `SkillDetail`
    
- Tutorial：`GET /tutorials/{id}` → `TutorialDetail`
    

**Tabs（按 type 组合）**

- Overview（所有）：`content.* + 关键字段摘要`
- Prompt：Content（prompt_text）+ Showcase（showcases）
- MCP：HowTo（howto.*）+ Cases（cases）
- Skill：Cases（cases）+ Files（zip_asset_id/媒体）
- Tutorial：Content（body_markdown）+ Media（media）

**Block D3 右侧元信息栏（轻动作）**

- 展示：作者/发布时间/统计
- 点赞：`PUT/DELETE /contents/{id}/vote`（返回 my_vote 当前态）
- 收藏：`PUT/DELETE /contents/{id}/favorite`（返回 favorited 当前态）
- 分享：前端生成分享卡/复制链接（可选：写 `share_open` event）

**Block D2-2 评论区（固定置底）**

- `GET /contents/{id}/comments` → CommentNode 树（desc）
- `POST /contents/{id}/comments`
- `DELETE /comments/{comment_id}`

---

### 3.4 编辑/发布页（/type/new, /type/{id}/edit）

**Block E2 表单**

- Prompt 必填：`title + category_id(标签必选) + tag_ids(可选) + model_name + language + prompt_text + showcases>=1`
- MCP 必填：`title + category_id + source + provider + repo_url + howto + cases>=1`
- Skill 必填：`title + category_id + source + provider + cases>=1`；`repo_url` 为空则 `zip_asset_id` 必填
- Tutorial 必填：`title + category_id + body_markdown + media`

**Block E4 按钮区（普通区域）**

- 保存：`PATCH /{type}/{id}`（保存后状态→PendingReview）
- 提交审核：`POST /contents/{id}:submit_review`
- 上架/下架：`POST /contents/{id}:list / :unlist`
- 删除：`POST /contents/{id}:delete`（物理删除，需二次确认）

---

### 3.5 收藏页（/me/favorites）

**左侧展开文件夹树**

- `GET /me/favorite-folders`（树 ≤5 层）

**右侧当前文件夹内容卡片流**

- 建议补 API：`GET /me/favorites?folder_id=xxx&offset&limit&sort&order&q`
- 返回：`ContentSummary[] + meta`

**删除文件夹**

- `DELETE /me/favorite-folders/{id}`
- 前端二次确认提示：删除后会取消该文件夹内全部收藏

---

### 3.6 我发布的内容（/me/published）

**左侧：发布 → 类型导航（展开）**

- Prompt/Skill/MCP/教程 四项（切换仅影响右侧列表）

**右侧列表（每个类型）**

- 复用对应类型列表接口：`GET /prompts|skills|mcps|tutorials`
- 固定 query：`author_id=me`
- 支持：`status + q + sort/order + offset/limit + tag_ids`

## 4) 空态 / 错态 / 加载态（按模版位置落位）

### 4.1 首页（/）

- 未搜索：结果区显示引导空态（提示“输入关键词开始全站搜索”）
- 无结果：空态（提供“清空关键词”）
- 加载：搜索结果 skeleton（卡片占位）
- 错误：重试按钮（不影响搜索框）

### 4.2 列表页

- 加载：左侧栏 skeleton + 卡片 skeleton
- 空态：
    - 过滤/搜索导致无结果：提示“无匹配结果”，提供“清空筛选/清空搜索”
    - 真空：提示“暂无内容”，若已登录可显示“去发布”
- 错误：
    - 401：跳 /login
    - 403：权限不足（通常只在非 Listed/私有列表场景出现）
    - 5xx：重试

### 4.3 详情页

- 加载：首屏 skeleton + Tabs skeleton + 评论 skeleton
- 401：跳登录（returnTo=当前）
- 403：权限不足页
- 409(ERR_STATE)：Toast + 刷新详情

### 4.4 编辑页

- 400(ERR_VALIDATION)：字段级错误 + 顶部汇总
- 401/403：按统一拦截
- 409(ERR_STATE)：提示“状态已变化，请刷新后重试”

### 4.5 个人中心

- 文件夹为空：右侧空态“暂无收藏”
- 发布列表为空：右侧空态“暂无发布内容”
- 删除文件夹：二次确认弹窗

### 4.6 后台

- 表格加载/空态/错误：标准表格态
- 401：跳登录；403：权限不足（无后台权限）

## 5) 权限拦截与回跳（路由级规则）

### 5.1 需登录页面（守卫）

- `/{type}/new`
- `/{type}/{id}/edit`
- `/me/**`
- `/admin/**`

**行为**

- 任何接口 401 → `redirect(/login?returnTo=<current_url>)`

### 5.2 403 行为

- 已登录返回 403：展示“权限不足”（不跳登录）

### 5.3 登录成功回跳

- 有 `returnTo` → 跳回
- 无 `returnTo` → `/`（首页）

## 6) 与 API 契约的差异（需要补齐/调整）

### 6.1 补充：全站搜索 API（支持混合卡片）

- `GET /search?q=&offset=&limit=&highlight=1&sort=&order=&tag_ids=&source=&type?`
- 返回：`ContentSummary[] + meta`
- 说明：`type` 可选；为空表示混合（按相关性/热度排序）

### 6.2 补充：个人收藏内容列表 API

- `GET /me/favorites?folder_id=...&offset&limit&sort&order&q`
- 返回：`ContentSummary[] + meta`

### 6.3 前台标签筛选（UI 合并）

- 前台只用 `tag_ids` 做筛选
- `category_id` 仍用于发布/编辑必填与后台管理（数据层不变）
